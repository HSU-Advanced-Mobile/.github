## 📱 Routine Master

**개인 맞춤형 홈케어 루틴 앱**으로, 사용자의 피부 타입·트러블·얼굴형 데이터를 바탕으로  
**아침/저녁 케어루틴**, **괄사 마사지 가이드**, **성분 추천**, **캘린더 기반 루틴 관리** 기능을 제공합니다.

- **사용자 피부 상태에 맞춘 체계적인 홈케어 루틴 제공**
- 사용자 피부 프로필 기반 **개인화된 가이드 및 추천 성분 안내**

---

## 🎯 프로젝트 주요 목표

- 사용자 피부 프로필 기반 **맞춤형 홈케어 루틴 자동 생성**
- 괄사 마사지 포함 **단계별 시각적 가이드 제공**
- 캘린더 기반 **루틴 실천 기록/조회 기능**

---

## 🧩 기능 요약

🧬 1. 개인 피부 프로필 관리

🧖‍♀️ 2. 맞춤형 홈케어 루틴 제공

🪬 3. 맞춤형 괄사 마사지 가이드

📆 4. 피부 관리 캘린더
